Приведение типов
    *Явное
    *Неявное
        *(type)value /C-style
        *С++
            *dynamic_cast<...>       // приводит класс родителя к наследнику и наоборот
            *static_cast<...>        // не умеет снимать константность, приводит тип на этапе компиляции
            *const_cast<...>         // приведение константности
            *reinterpret_cast<...>   // позвоялет делать практически любые изменения типов
                                     // например из double в int*
            *bit_cast<...>           // меняет чтение битов объекта
